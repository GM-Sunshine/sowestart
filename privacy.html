Privacy Policy

Last Updated: November 23, 2025

Privacy First: So We Start is designed with your privacy in mind. We do not collect, store, or transmit any personal data to external servers.

Overview

So We Start ("we", "our", or "the extension") is a Chrome browser extension that provides a customizable new tab page. This privacy policy explains how we handle your information.

Data Collection and Storage

We do NOT collect or transmit your personal data. All data is stored locally on your device using Chrome's storage API. Specifically:

  • Settings: Theme preferences, clock format, language selection
  • Quick Links: Your saved bookmarks and custom links
  • Background Preferences: Your selected background type and search queries
  • Weather Location: City name (if manually entered) or cached IP-based location
  • To-Do Items: Your personal tasks and notes

Third-Party APIs

So We Start uses the following external APIs to provide functionality. These services may collect data according to their own privacy policies:

  • Sourcesplash (sourcesplash.com) - Background images from Pexels, Pixabay, and Unsplash
  • Picsum Photos (picsum.photos) - Random background photos
  • Art Institute of Chicago API (api.artic.edu) - Public domain artwork images
  • Open-Meteo (open-meteo.com) - Weather data (no API key required, privacy-friendly)
  • ipwho.is (ipwho.is) - IP-based geolocation for automatic weather location detection

Chrome Sync (Optional)

If you enable Chrome Sync in your browser settings, your So We Start preferences may be synchronized across your devices using Google's secure sync service. This is handled entirely by Chrome and is subject to Google's Privacy Policy.

Permissions Explained

So We Start requests the following Chrome permissions:

  • storage: To save your settings, links, and preferences locally on your device
  • bookmarks: To allow you to import bookmarks from Chrome (only when you explicitly request it)
  • Host permissions: To fetch background images, weather data, and artwork from the APIs listed above

Data Sharing

We do not share, sell, or transmit any of your data. All information stays on your device. We have no servers, no analytics, and no tracking.

Data Export and Deletion

You have complete control over your data:

  • Export: Use Settings → Data → Export Settings to download your data as a JSON file
  • Import: Restore your data from a backup file
  • Reset: Use Settings → Data → Reset to delete all stored data
  • Uninstall: Removing the extension will delete all locally stored data

Children's Privacy

So We Start does not knowingly collect any information from children under 13. The extension is suitable for all ages.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this page.
